## 命名
### 變數
變數要小寫，若有多個單字組成，用底線分開來。
```
student_name = "Mary"
```
### 常數
常數要大寫。
```
PI = 3.1415926
```

### 函式
函式要小寫，若有多個單字組成，用底線分開來。

### 類別
類別用大駝峰。
```
StudentInfo
```
